Ето базовые стили используемые для валидации формы

Input style : {{ myInput.className }}

Обратить внимание на изменение классов input, при следуйших действиях

  1. Кликнуть на поле ввода а потом за его пределами ( ng-untouched => ng-touched )
  2. Ввести значение в поле ввода ( ng-pristine => ng-dirty )
  3. Очистить поле ( ng-valid => ng-invalid )
